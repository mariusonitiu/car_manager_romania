# Car Manager România

Integrare custom pentru Home Assistant dedicată administrării unuia sau mai multor autovehicule.

## Funcționalități planificate

- Evidență revizii și mentenanță
- Intervale service (ulei, filtre, distribuție etc.)
- Rovinietă (integrare existentă)
- RCA / asigurare auto
- ITP
- Notificări automate
- Istoric intervenții
- Card Lovelace dedicat
- Integrare OBD (ESP32 + MQTT)

## Stadiu proiect

🚧 În dezvoltare

## Instalare (manual)

1. Copiază folderul `car_manager_romania` în:
/config/custom_components/

2. Restart Home Assistant
3. Adaugă integrarea din UI
